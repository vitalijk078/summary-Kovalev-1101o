1.Типы данных
Число (Number) Свойства - Number.MAX_VALUE, Number.MIN_VALUE. Методы - .toString(), .toFixed().
Строка (String) свойства - length методы - .toUpperCase(),.toLowerCase().
Булево значение (Boolean) метод - .valueOf()
Объект (Object) свойства и методы определяются объектом
Массив (Array) свойства - length методы - .push(), .pop(), .shift(), .unshift()
2.Структуры данных
Массив (Array) var arr = []
Стек (Stack) var stack = [],stack.push(element), stack.pop()
Очередь (Queue) var queue = [],queue.push(element),queue.shift()
3.Арифметические операции
Сложение (+) вычитание (-)
умножение (*) деление (/)
Остаток от деления (%) возведение в степень (**)
4.Функции из библиотеки Math
Math.abs() возвращает абсолютное значение числа
Math.floor() округляет число вниз
Math.ceil() округляет число вверх
Math.random() возвращает случайное число от 0 до 1
5.Логические операции:
И (&&)
ИЛИ (||)
отрицание (!)
6.Алгоритмические конструкции:
Условие (if-else) if (условие) { блок кода } else { блок кода }
Цикл(for) for (начальное_значение; условие; шаг) { блок кода }
Цикл(while) while (условие) { блок кода }
Цикл(do-while) do { блок кода } while (условие)
7.Функции
Объявление функции function functionName() { блок кода }
Стрелочные функции const functionName = () => { блок кода }
Импорт и экспорт функций export function functionName() { блок кода } import { functionName } from 'модуль'
8. Функции lodash
_.forEach( collection, iteratee) выполняет функцию для каждого элемента коллекции
_.map( collection, iteratee) создает новый массив с результатами вызова функции для каждого элемента коллекции
_.filter( collection, predicate) создает новый массив с элементами, для которых функция возвращает true
